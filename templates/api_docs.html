Documentación API v1

Integra tu sistema con nuestra plataforma de recargas

Autenticación

Todas las peticiones requieren tu API Key en el header:

X-API-Key: TU_API_KEY_AQUI

Obtén tu API Key en Mi API. También puedes enviarla como parámetro ?api_key=TU_KEY

URL Base

{{ request.host_url }}api/v1/
GET

/api/v1/saldo

Consulta el saldo disponible en tu cartera.

Ejemplo de petición:

curl -H "X-API-Key: TU_KEY" {{ request.host_url }}api/v1/saldo

Respuesta:

{
  "ok": true,
  "saldo": 150.00,
  "nombre": "Tu Nombre"
}
GET

/api/v1/productos

Lista todos los productos disponibles con sus precios.

Respuesta:

{
  "ok": true,
  "productos": [
    {
      "id": 1,
      "nombre": "100+10 Diamantes",
      "descripcion": "Recarga de 110 diamantes Free Fire",
      "precio": 1.50,
      "categoria": "Free Fire"
    }
  ]
}
POST

/api/v1/comprar

Realiza una compra. Para recargas de Free Fire, el canje se ejecuta automáticamente.

Parámetros (JSON Body):

CampoTipoRequeridoDescripción
producto_idintegerSíID del producto a comprar
cantidadintegerNoCantidad (default: 1)
id_juegostringFree FireID del jugador en el juego

Ejemplo de petición:

curl -X POST \
  -H "X-API-Key: TU_KEY" \
  -H "Content-Type: application/json" \
  -d '{"producto_id": 1, "id_juego": "123456789"}' \
  {{ request.host_url }}api/v1/comprar

Respuesta exitosa (Free Fire):

{
  "ok": true,
  "pedido_id": 42,
  "total": 1.50,
  "saldo_restante": 148.50,
  "nombre_jugador": "homer0.27",
  "mensaje": "Pedido #42 completado. Recarga aplicada a homer0.27 (ID: 123456789)"
}

Respuesta error (saldo insuficiente):

{
  "ok": false,
  "error": "Saldo insuficiente",
  "saldo": 0.50,
  "total": 1.50
}
